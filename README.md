# Создание информационной системы "Волонтерская деятельность"

Сайт создавался в качестве курсовой работы. Включает в себя админ-панель, возможность создавать профили организаторов и волонтеров, подавать заявления на мероприятия и в качестве отчета выводить волонтерскую книжку

# Инструкция

1) Необходимо скачать базу данных и импортировать ее на сервер (я использовала OpenServer)

2) Создать аккаунт или войти в существующий

3) Выбрать необходимое мероприятие

4) Ожидать пока организатор одобрит заявку
